## Quick Start

Get up and running with CASSIE MARIE | AUTHOR in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://cassiemarieauthor.com` with your email. Verify your account to unlock full features.

  </Step>
  <Step title="Set Up Project" icon="folder-plus">
    Create a new documentation space. Add your first page using the intuitive editor.

  </Step>
  <Step title="Add Content" icon="edit-3">
    Write in Markdown or rich text. Use the preview to see live changes.

    <CodeGroup tabs="Markdown,Rich Text">

      - Feature one
      - Feature two

      </ul>
