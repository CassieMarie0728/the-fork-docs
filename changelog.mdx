## Upgrading to the Latest Version

Follow these steps to upgrade your CASSIE MARIE | AUTHOR installation.

<Steps>
  <Step title="Backup Your Data" icon="database">

  Create a backup of your current configuration and sessions.

  ```bash
  cp -r ./config ./config-backup
  cp -r ./sessions ./sessions-backup
  ```

  </Step>
  <Step title="Update Package" icon="download">
    
    <Tabs>
      <Tab title="npm" icon="package">
      ```bash
      npm install cassie-marie-author@latest
      ```
      </Tab>
      <Tab title="yarn" icon="package">
      ```bash
      yarn add cassie-marie-author@latest
      ```
      </Tab>
    </Tabs>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Apply any database migrations.

    <CodeGroup tabs="PostgreSQL,MySQL">
    ```sql
    -- PostgreSQL
    npm run migrate:up
    ```
    ```sql
    -- MySQL
    npm run migrate:mysql
    ```
    </CodeGroup>

  </Step>
  <Step title="Restart Server" icon="refresh-cw">

    Restart your application server.

    ```bash
    npm run start:prod
    ```

  </Step>
</Steps>

<Callout kind="alert" title="Breaking Changes">
Review the `v1.0.0` breaking changes if upgrading from an earlier version. Update your `SESSION_SECRET` environment variable.
</Callout>
