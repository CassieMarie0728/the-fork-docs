## Prerequisites

<Callout kind="info" title="Before You Begin">
Ensure you have:
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An email address for account verification
- Optional: GitHub account for repository integration
</Callout>

## Create Your Account

Follow these steps to register and log in to CASSIE MARIE | AUTHOR.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Navigate to `https://dashboard.example.com` and click **Sign Up**.
  </Step>
  <Step title="Enter Your Details" icon="user">
    Provide your email, create a password, and complete the CAPTCHA.
    
    <Callout kind="tip">
      Use a strong password with at least 12 characters, including uppercase, lowercase, numbers, and symbols.
    </Callout>
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification link and click it to activate your account.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard, enter your credentials, and access your workspace.
  </Step>
</Steps>

## Create Your First Documentation Space

Once logged in, set up a new space for your project.

<Steps>
  <Step title="Click New Space" icon="plus">
    From the dashboard, select **New Documentation Space**.
  </Step>
  <Step title="Configure Space" icon="settings">
    Enter a name like `My Project Docs`, select visibility (Public or Private), and choose a template if available.
  </Step>
  <Step title="Save and Open" icon="save">
    Click **Create**. Your space opens with a default structure.
  </Step>
</Steps>

## Upload Initial Documents

Add your first files using one of these methods.

<Tabs>
  <Tab title="Web Upload" icon="upload">
    Drag and drop Markdown, PDF, or image files directly into the editor.
    
    <Image
      src="https://via.placeholder.com/800x400?text=Web+Upload+Interface"
      alt="Web upload interface showing drag-and-drop area"
      width="800"
      height="400"
    />
  </Tab>
  <Tab title="GitHub Sync" icon="github">
    Connect your repo:
    
````bash
# Add repo URL in settings
https://github.com/your-username/the-fork
````
    
    Sync pulls MDX files automatically.
  </Tab>
  <Tab title="CLI Upload" icon="terminal">
    Install the CLI and upload:
    
    <CodeGroup tabs="npm,yarn">
````bash
npm install -g cassie-cli
````
````bash
yarn global add cassie-cli
````
    </CodeGroup>
    
````bash
cassie-cli login --email your@email.com
cassie-cli upload ./docs --space "My Project Docs"
````
  </Tab>
</Tabs>
